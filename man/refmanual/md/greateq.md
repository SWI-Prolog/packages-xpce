# class greateq {#class-greateq}

Conditional expression.  Evaluates both left and right sides and compare
the result.  See class binary_condition for details.

@see topic Conditions


## Send methods {#class-greateq-send}

- greateq->_execute
    Evaluates <-left and <-right to a numeric value and succeed if <-left
    evaluates to a larger than or equal value than <-right.
